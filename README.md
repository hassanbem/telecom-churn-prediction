# 📊 Customer Churn Prediction — End-to-End ML Project

## 🚀 Overview

This project predicts customer churn in the telecommunications industry using machine learning models.
The goal is to identify customers at risk of leaving and help businesses improve retention strategies.

---

## 🧠 Project Pipeline

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training & Evaluation
* Model Deployment using Flask API

---

## 📊 Results

* **F1 Score:** 94%
* **Recall:** 93%
* Strong performance on imbalanced dataset

---

## 🛠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn, XGBoost
* Flask (API deployment)

---

## ⚙️ How to Run

```bash
git clone https://github.com/hassanbem/customer-churn-prediction.git
cd customer-churn-prediction

pip install -r requirements.txt
python app/app.py
```
